# Tuts Plus: 30 Days to learn jQuery

Learning jQuery with Tuts Plus

## Lesson 1
* jQuery is function
* exposed as global on window
* aliased as $
* can pass it css selectors
* used to 'query' and manipulate DOM

## Lesson 2

* Need to tell jQuery to wait until document is ready
